# SEESO

## 説明
専門学校岡山情報ビジネス学院(OIC)の学生専用のイベント作成用SNSです。<br>
SEESOの由来は、<br>
<br>
S・・・stusent(学生)<br>
E・・・event(イベント)<br>
E・・・enjoy(楽しむ)<br>
S・・・SNS(交流)<br>
O・・・OIC(専門学校岡山情報ビジネス学院)<br>
<br>
となっています。<br>
<br>
※現在作成中

## 機能
- リアルタイムチャット機能
- イベント作成機能

## 使用言語・ツール
- HTML
- CSS
- JavaScript
- Firebase(Authentication、Cloud Firestore、Hosting)
